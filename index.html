Common Questions

Why charge for a Terminal command?

You're paying for convenience, not the command. The command is free and we share it openly. TouchBarFix provides a one-click solution without needing to open Terminal.

What if it doesn't work?

We offer a 30-day money-back guarantee. If TouchBarFix doesn't solve your issue (note: it only fixes software freezes, not hardware problems), email us for a full refund.

Is it safe?

Yes. The app is notarized by Apple, doesn't require admin privileges, and the source code is available on GitHub for review. It only restarts Touch Bar processes, nothing more.

Which MacBooks are supported?

All MacBook Pro models with Touch Bar (2016-2021) running macOS Big Sur (11.0) or later. The app is a universal binary that runs natively on both Intel and Apple Silicon.
