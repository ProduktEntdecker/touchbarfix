Privacy Policy

Last updated: 2025-08-20

This Privacy Policy describes how TouchBarFix ("we", "us") collects and uses information when you visit our website and use our software.

Summary

  • We collect minimal data. The app works without personal data.
  • Email is collected only if you explicitly submit it (e.g., to receive a download link/updates).
  • We use privacy-friendly analytics (no cookies, no tracking across sites).
  • We do not sell personal data.

Data We Collect

  • Email (optional): Provided by you via a form on the website. Metadata may include timestamp, page URL (source), basic user agent (for diagnostics), and consent status.
  • Web analytics: Page views and events via Plausible Analytics (aggregated, no cookies).
  • Server logs: Basic request logs for security and debugging (no PII where feasible).

Purpose and Legal Basis

  • Deliver requested content (e.g., send download link) – contract/performance.
  • Product updates and important notices (optional, with consent) – consent.
  • Security, abuse prevention, and service reliability – legitimate interests.

Retention

  • Emails: kept until you unsubscribe or request deletion.
  • Logs/analytics: short, rolling periods sufficient for debugging and metrics.

Processors and Infrastructure

  • Hosting: Vercel (static site + serverless functions)
  • Code hosting/releases: GitHub
  • Email delivery (if configured): Resend/Postmark/SES
  • Analytics: Plausible (privacy-friendly, no cookies)

Your Rights

Where applicable (e.g., EU/UK), you may request access, rectification, deletion, restriction, portability, or object to processing. You can withdraw consent at any time.

Security

  • HTTPS everywhere, signed/notarized binaries, least-privilege entitlements.
  • Server-side input validation and basic rate limiting for forms.
