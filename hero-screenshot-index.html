Yes. TouchBarFix is notarized by Apple, doesn't require admin privileges, and only restarts Touch Bar processes. The source code is on GitHub for review.

You absolutely can! The command is: pkill -x TouchBarServer. TouchBarFix just makes it more convenient with one-click access without Terminal.
