Help & Support

Last updated: 2025-08-20

Install

  1. Download the DMG from the website or GitHub Releases
  2. Open the DMG and drag TouchBarFix.app to Applications
  3. First launch: Right‑click the app → Open → Open (Gatekeeper prompt)

Use

  • Open TouchBarFix and click “Fix Touch Bar”
  • Wait ~2 seconds; Touch Bar processes will restart
  • If needed, click again. Persistent issues may be hardware‑related

Tips

  • Works on Touch Bar MacBook Pro models (2016–2021)
  • macOS 11.0+ recommended; see README for supported versions

Common Issues

  • App won’t open: Ensure you used Right‑click → Open for the first run
  • Touch Bar still unresponsive: Run the fix again; reboot as a last resort
  • Dock flashes: The app may restart Dock when resetting Touch Bar preferences
